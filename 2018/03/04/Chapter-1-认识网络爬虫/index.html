Chapter-1 认识网络爬虫

发表于 2018-03-04 | 分类于 爬虫相关 ， chapter 01

1-1认识网络爬虫

from urllib.request import urlopen

urllib 是Python 的标准库（就是说你不用额外安装就可以运行这个例子），包含了从网络请求数据，处理cookie，甚至改变像请求头和用户代理这些元数据的函数。

html = urlopen("http://pythonscraping.com/pages/page1.html")
print(html.read())

下面是整个页面的html数据:

b'<html>\n<head>\n<title>A Useful Page</title>\n</head>\n<body>\n<h1>An Interesting Title</h1>\n<div>\nLorem ipsum dolor sit amet,consectetur adipisicing elit,sed do eiusmod tempor incididunt ut laboreet dolore magna aliqua. 1-2 使用beautifulsoup4
from bs4 import BeautifulSoup # 导入模块
使用beautifulsoup来查看<h1>内容
bsObj = BeautifulSoup(html.read()) 
print(bsObj.h1)

会运行几秒,我们可以看到提示：

# <h1>An Interesting Title</h1>
#html内容就被取出来了

此外还有可以提取网站的其他内容

print(bsObj.div)
print(bsObj.body)
