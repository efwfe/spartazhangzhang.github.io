发表于 2018-03-31 | 分类于 web data

转化率进阶

购买前访问天数

  如果购买前天数比较长，掌握好时间节点能够向用户传递信息，促使其作出最后的购买决定。如果购买前天数比较短。说明消费具有冲动型，可以进行价格调整策略。

平均订单价值

  如果只是看转化率可能出现的情况是，转化率很高但是订单数减少了，这个时候可以考虑使用销售额来平衡局面，不过还有更好的办法是施工平均订单价值Average Order Value（总销售额除以订单数）来分析。

目标群体

  了解访客的目的，然后才能计算出真正潜在的客户数，只是极可能产生转化的可数数目，可以通过线上访客的关注内容进行分析，当然还有线下点调研。

宏观转化和微观转化

从整体着眼，考察宏观转化，对于精彩部分，着重标识和衡量微观转化的衡量。

微观转化对了解网站目的以外的其他功能。

  衡量网站的多元化影响；

  强制你了解访客在站内的格式行为，对用户细分，行为研究，绩效分析帮助很大；

  当然也能打开思路，带来更多的思考空间。

衡量非电子商务网站的绩效

  1.访客的忠诚度，能够帮助分析掌握用户行为；

  2.访客的回访率，可以显示用户是否接收到了新的内容；

  3.网站停留时间，其实就是用户的黏性，虽然停留时间在一定程度上会有出入，但是对用户分类分析起到很大的作用；

  4.访客的访问深度，将标准访问深度报表和停留时间，访问间隔，搭配来看的话能够有助于改善受众和业务的发展；

对于非电商网站必须明确何种用户行为是业务价值所在，然后再去考察它。
